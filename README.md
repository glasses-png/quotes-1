## Behaviour Driven Devlopment(BDD)

| Behaviour                |  Input                | Output                 |
| -------------            | -------------         |------------------------|
| Submit                   | Quote,Author,date     | adds Quote             |
| -------------            | -------------         |------------------------|
| Upvote                   |                       | adds quotes votes      |   
| -------------            | -------------         |------------------------| 
| Downvote                 | lengths(3,4,5)        | subtracts quotes votes |
| -------------            | -------------         |------------------------|
| Delete                   |                       | deletes quote          |
| -------------            | -------------         |------------------------|
